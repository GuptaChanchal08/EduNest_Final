# EduNest Backend API

Express.js REST API for the EduNest online learning platform.

## Base URL
`http://localhost:4000/api/v1`

## API Routes

- `/auth` тАФ Register, Login, OTP, Password Reset
- `/profile` тАФ User profile management
- `/course` тАФ Course CRUD, sections, subsections
- `/payment` тАФ Razorpay payment integration
- `/reach` тАФ Contact form
- `/admin` тАФ Admin panel (Admin only)
